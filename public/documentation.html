Movie Info Page

all the information on thes pages is sourced through an api.
this api is built upon REST architecture. the database will be built with MongoDB.
the business logic will be constructured mongoose.
the data from the api will be provided in json format.
it will require user authentication.
all the source code will be deployed publicly on github.

URL Description
/ This is the primary homepage of the applicaton
/books appending this will bring you to a json of all the data we have stored
/Documentation.html this will lead to all Documentation
/books/:title gets specific books details through title
Post /books Posts new book information to list of books assuming the criteria are met
Delete /books/:id deletes specific book from list based upon the books assigned ID
